## Deployment

Install dependencies
```bash
 npm install
```

To deploy this project run
```bash
  npm run deploy
```
Then run npm run dev to run the project on the browser.
```bash
  npm run dev
```

Runs the app in the development mode.
Open **http://localhost:5173** to view it in your browser.
